# Model Assumptions

1. The model uses a daily time step.
2. Forecasting is simulation-based, not machine-learning based.
3. Residual waste is calculated from incoming waste, treated waste, and diverted waste.
4. Net Landfill Load equals residual waste after restriction adjustment.
5. Accumulated waste is treated as the main stock.
6. Remaining capacity is derived from total landfill capacity minus accumulated waste.
7. Since the landfill is reported as overloaded, overload accumulation is used as an important output.
8. Fuzzy AHP risk index is used as a risk-informed interpretation layer, not as the main Net Landfill Load equation.
9. Scenarios are policy assumptions and can be adjusted after team validation.
